## 2. 개체 디지털 트윈 (5분) — AI가 소를 읽는다

**화면**: `/cow/{id}` (423번 등 이벤트 풍부한 개체)

### 2-1. 개체 프로필
- 이력번호, 품종, 산차, 번식상태
- smaXtec 센서 데이터 + 축산물이력추적시스템(data.ekape.or.kr) 공공데이터 연동
- 트래픽라이트 배지: 건강(녹), 주의(황), 위험(적)

### 2-2. 30일 센서 차트
- 체온/반추/활동 트렌드 (CollapsibleCard — 모바일 접힘, 데스크톱 펼침)
- 3단계 폴백: smaXtec live → sensor_daily_agg → sensor_measurements hourly
- AI 건강 점수 (50~100점 범위)

### 2-3. 소버린 AI 버튼
- 클릭 → Claude AI가 실시간으로 개체 상태 해석
- "이 소의 현재 상태는... 추천 조치는..."
- **핵심 메시지**: "한국 데이터는 한국 AI가 해석합니다 (소버린 AI)"

### 2-4. 백신 접종이력 + 방역검사
- 스크롤 → 백신 접종이력 섹션
- 국가 접종 이력: 이력추적시스템(data.go.kr) 실데이터
- 방역검사 결과: 브루셀라 음성, 결핵 음성
- "공공데이터포털의 백신+검사 이력이 개체 프로필에 실시간 통합됩니다"

### 2-5. 감별진단 (수의사 핵심 기능) ⭐
- **화면**: `/cow/{id}` 프로필 하단 "🔬 감별진단" 카드
- "감별진단 실행" 버튼 클릭 → API가 센서·이력 데이터 분석
- **결과 표시**:
  - 긴급도 배너 (즉시/24시간/정기) + 데이터 품질 표시
  - 6개 후보 질병 확률 바 (유방염/케토시스/산독증/폐렴/자궁내막염/제4위변위)
  - 각 질병 펼치면: 센서 근거(✓지지/✗반증/—중립) + 확인검사 목록
  - 농장 90일 진단 이력 바 차트
- **시연 포인트**: "smaXtec 알람 → CowTalk AI 감별진단 → 확인검사 안내 → 치료 기록 — 행동 연결 완결"
- **추천 멘트**: "AI가 수의사의 임상적 사고를 구조화합니다. 센서 데이터 + 이력을 종합해 가장 가능성 높은 질병부터 순서대로 보여줍니다."

### 2-6. 이벤트 타임라인
- 발정/수정/임신감정/질병 이벤트 히스토리
- "센서 알람 → AI 판단 → 행동 추천 → 기록"이 한 화면에서 완결

## 3. 팅커벨 AI 대화 (3분) — 대화로 소를 관리하는 시대

**화면**: 대시보드 또는 개체 상세 페이지 하단 TinkerbellAssistant 바

### 3-1. 자연어 질의 시연
팅커벨에게 직접 물어보기 (추천 질문):
- "423번 소 최근 상태 알려줘" → `query_animal` + `query_animal_events` 도구 자동 실행
- "해뜨는목장 요약해줘" → `query_farm_summary` + `get_farm_kpis` 도구 실행
- "오늘 발정 감지된 소 있어?" → `query_breeding_stats` 도구 실행

### 3-2. 도구 실행 추적
- AI가 어떤 도구를 호출하는지 화면에 표시
- "AI가 스스로 판단해서 20개 도구 중 적합한 것을 선택합니다"
- 도구 목록: 개체 조회, 센서, 번식, 방역, 공공데이터, 기상, 치료 기록 등

### 3-3. 감별진단 대화 시연
- "423번 감별진단 해줘" → `query_differential_diagnosis` 도구 자동 실행
- AI가 6개 후보 질병 + 확률 + 센서 근거 + 확인검사를 자연어로 설명
- "AI가 수의사의 진단 사고 과정을 대화로 안내합니다"

### 3-4. 행동 연결 시연
- "423번 수정 적기 추천해줘" → `recommend_insemination_window` 도구
- AI가 수정 적기 + 추천 정액까지 제시
- "AI가 알람에서 행동까지 연결합니다"

### 3-5. 음성 입력 (옵션)
- 마이크 버튼 → 음성으로 질문 → Web Speech API
- "현장에서 목장주가 손 대신 음성으로 조작할 수 있습니다"

## 4. 번식 관리 (3분) — 발정→수정→임신 루프

### 4-1. 번식 캘린더
**화면**: `/breeding/calendar`
- 월간 7x5 그리드 + 주간 리스트 뷰 전환
- 8종 이벤트: 발정예정/수정/임신감정/건유/분만 등
- 긴급도별 색상: critical(적)/high(주)/medium(청)/low(회)
- 농장 필터, 개체 클릭 → 개체 상세 이동
- "수의사와 수정사의 일정표가 자동으로 만들어집니다"

### 4-2. 번식 KPI
**화면**: `/breeding/performance`
- 수태율, 발정탐지율, 평균공태일, 분만간격, 첫수정일수
- 6단계 파이프라인: 대기→발정→수정→임신→후기→분만
- "목장 번식 성적을 숫자로 관리합니다"

### 4-3. 센서 비교 뷰 (옵션)
**화면**: `/sensor/compare`
- 최대 4두 동시 비교 (체온/활동/반추 오버레이)
- earTag 검색, 기간 토글 (2/7/30일)
- 38.5도 발열 기준선 표시
- "같은 우군의 소들을 한눈에 비교합니다"

## 5. 방역관 대시보드 (4분) — 146농장 통합 방역

**역할 전환**: 방역관(quarantine@test.kr / test1234)
**화면**: `/epidemiology/dashboard` (자동 리다이렉트)

### 5-1. 방역 KPI + 접종현황
- 위험등급 배너, 6개 KPI 카드
- 법정 백신 접종현황: 구제역/브루셀라/럼피스킨/소유행열별 접종률
- "146개 농장의 방역 상황을 한 화면으로"

### 5-2. 전국 현황 지도
**화면**: `/epidemiology/national` 또는 대시보드 내 NationalMiniMap
- 시도별 농장 마커 + 발열 클러스터 표시
- 마커 클릭 → 농장 상세 드릴다운
- "전국 축산 현황을 지도 한 장으로"

### 5-3. 지역 인텔리전스 지도
**화면**: `/regional-map`
- 시도 경계선 + 비례 마커 (두수 비례)
- 필터: 농장상태 / 발정활동 / 건강위험 / 센서활성도
- 마커 클릭 → `/farm/:farmId` 농장 상세 이동

### 5-4. 접촉 네트워크 + 확산 시뮬레이션 (시간 여유 시)
- `/epidemiology/contact-network` — 농장 간 이동 이력 기반 접촉 그래프
- `/epidemiology/simulation` — SEIR 모델 기반 확산 예측
- "구제역 발생 시 30초 내 전파 경로 파악 + 피해 규모 사전 시뮬레이션"

### 5-5. 팅커벨 방역 모드
- 방역관 역할 + 방역 키워드 → 자동 방역 모드 활성화
- 방역 전용 suggestions, 오렌지 테마
- "경기도 발열 현황 알려줘" → `query_national_situation` 도구 + 시도별 데이터

## 7. 클로징 (2분)

- "smaXtec 센서 + CowTalk AI = 글로벌 축산 디지털 전환 패키지"
